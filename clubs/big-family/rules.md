# Big Family — правила для бота

Клуб: **Big Family** (собственник — Антон).

## Формат дня 1

1. **Ночь знакомства** — по стандартным правилам клуба (если играете с ночью).
2. **Представление** — по **1 минуте** на игрока, круг **с места №1** по часовой.
3. **Сразу голосование** — без отдельного дневного круга речей: выставление → голосование → ночь.

## Что делает бот

- При «Знакомство» автоматически стартует с **№1** (без выбора первого).
- Таймер представления: **60 секунд**.
- После последнего представления — кнопки «Выставить» и «Голосование», без «Кто начинает день».

## Экономика клуба (для продаж)

- Вход ~**1 000 ₽** с игрока.
- Типичный стол: **10–12** человек → **10–12 тыс. ₽** за вечер.
- Тариф **Mini 3 499 ₽/мес** (10 игр, 1 ведущий) окупается с **одной** игры.

## Подключение

1. Создать клуб в боте с названием вроде `Big Family` или `Биг Фэмили`.
2. Выполнить `apply_settings.sql` в Supabase (опционально — бот и так подхватит пресет по имени).
3. Провести тестовую игру с Антоном.

Исходные правила PDF: `Правила_игры_в_мафию_от_клуба_«Big_Family» (1).pdf` в корне репозитория.
