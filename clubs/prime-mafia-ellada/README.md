## Загрузка в бот

1. Выполнить `apply_settings.sql` для клуба в Supabase.
2. От имени админа: `/club_cards` → выбрать клуб Ellada.
3. Отправить каждую картинку с **точной** подписью роли из таблицы выше.
4. Проверить: `/club_roles_status`.

## Mini app

После `apply_settings.sql` у клуба включается тема **Ellada** (тёплая палитра как на картах).

Подробнее: `styling.md`, механика ролей: `roles.md`.
