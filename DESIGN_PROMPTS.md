# Prime Mafia — промты для дизайна

## Общий стиль

```text
Create a premium UI design concept for Prime Mafia, a Telegram Mini App and digital platform for Mafia game clubs.

Audience: Mafia hosts, club owners, organizers, and offline game communities.

Core value: automation for the host and club operations. The product helps create games, assign roles, run timers, manage night actions, track voting, store player history, ratings, residents, announcements, and club branding.

Visual style: dark elegant luxury, black and deep burgundy atmosphere, subtle gold accents, cinematic intellectual club mood, modern premium SaaS interface, readable typography, serious and mature. Not childish, not cartoon, not casino.

Device focus: iPad landscape first, then mobile Telegram Mini App.

Brand feeling: exclusive offline club, control, order, mystery, prestige, premium community.
```

## Виртуальный стол

```text
Design a Telegram Mini App screen for Prime Mafia showing a virtual Mafia table for a live offline game.

Layout: iPad landscape. A circular table in the center with 16 player seats around it. Each seat has a player number, name, status indicator, and small role/privacy-safe marker. In the center of the table: current phase, large timer, game code, and day/night indicator.

Right side: host control panel with buttons for next player, pause timer, start vote, go to night, open night actions, add foul, eliminate player.

Left side: compact game summary with club name, ruleset, players alive, residents, tourists, and current round.

Mood: premium dark club, elegant, clear, focused on host automation.

Avoid: cartoon mafia characters, casino chips, guns, childish icons, cluttered UI.
```
