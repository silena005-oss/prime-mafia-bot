## Публикация итогов в Telegram

**Технически — да.** Не нужна «привязка канала» через Telegram Login; достаточно `chat_id`.

| Способ | Как |
|--------|-----|
| Группа клуба | Бот добавлен в группу → собственник **пересылает** сообщение из группы боту → сохраняется `telegram_chat_id` |
| Канал | Бот — **администратор** канала с правом публикации → та же пересылка или `@channel` + id |
| После игры | Кнопка «📢 Отправить в группу клуба» или **автопубликация** в настройках клуба |
| Без автоматизации | «📣 Текст для публикации» — ведущий копирует в чат вручную |

Модуль: `lib/publikaciya.js`. Настройки: ⚙️ Настройки клуба → привязка группы.

Ограничения Telegram: бот не может писать в группу, если его выгнали; в канале нужны права admin.
